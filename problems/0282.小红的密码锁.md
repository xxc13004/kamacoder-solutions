[题目链接](https://kamacoder.com/problempage.php?pid=1362)
## 思路分析
按照数字的每一位依次计算步骤即可，如果当前位正确密码小于当前密码，直接相减即可，否则就+10再相减。
## 复杂度分析
1. 时间复杂度：O(1)；
2. 空间复杂度：O(1)。
## 代码实现

``` cpp
}
```

### Python
``` python
a, b = input().strip().split()
ans = 0
for i in range(4):
    if a[i] < b[i]:
        ans += ord(a[i]) + 10 - ord(b[i])
    else:
        ans += ord(a[i]) - ord(b[i])
print(ans)
```

### Java
``` java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String a = scanner.next();
        String b = scanner.next();
        int ans = 0;
        for (int i = 0; i < 4; i++) {
            if (a.charAt(i) < b.charAt(i)) {
                ans += a.charAt(i) + 10 - b.charAt(i);
            } else {
                ans += a.charAt(i) - b.charAt(i);
            }
        }
        System.out.println(ans);
    }
}
```
