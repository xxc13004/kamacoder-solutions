## 思路分析

根据题意可知，数字的凸包区间的下界和上界分别是当前出现的所有的数字的最小值和最大值。随着数字的数量增加，这个下界只会变的更小，上界只会变得更大。

1. 当下界比0大的时候，答案只会是0；
2. 当下界变成0的时候，答案就变成了**上界+1**。

## 复杂度分析
1. 时间复杂度：O(n)；
2. 空间复杂度：O(n)。

### Java
``` java
import java.io.*;
import java.util.*;

public class Main {
    private static final int MOD = 1000000007;
    
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        int n = Integer.parseInt(br.readLine());
        int[] nums = new int[n];
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        for (int i = 0; i < n; i++) {
            nums[i] = Integer.parseInt(st.nextToken());
        }
        
        int l = MOD, r = -1;
        for (int i = 0; i < n; i++) {
            l = Math.min(nums[i], l);
            r = Math.max(nums[i], r);
            
            if (l <= 0) {
                bw.write((r + 1) + " ");
            } else {
                bw.write("0 ");
            }
        }
        
        bw.flush();
        bw.close();
        br.close();
    }
}
```

### Go
``` go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    in := bufio.NewReader(os.Stdin)
    out := bufio.NewWriter(os.Stdout)
    defer out.Flush()
    
    var n int
    fmt.Fscan(in, &n)
    
    nums := make([]int, n)
    for i := 0; i < n; i++ {
        fmt.Fscan(in, &nums[i])
    }
    
    MOD := 1000000007
    l := MOD
    r := -1
    
    for i := 0; i < n; i++ {
        if nums[i] < l {
            l = nums[i]
        }
        if nums[i] > r {
            r = nums[i]
        }
        
        if l <= 0 {
            fmt.Fprint(out, r+1, " ")
        } else {
            fmt.Fprint(out, 0, " ")
        }
    }
}
```
