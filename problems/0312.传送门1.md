# 脑筋急转弯
[题目链接](https://kamacoder.com/problempage.php?pid=1392)
## 思路分析
因为可以任意顺序使用这些传送门，所以可以先使用所有传送值为负数的传送门，然后翻转，再使用所有正数的传送门。最终结果就是对数组的所有元素取绝对值相加即可。
## 复杂度分析
1. 时间复杂度：O(n)；
2. 空间复杂度：O(1)。

### Python
``` python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    nums = list(map(int, data[1:1+n]))
    
    ans = 0
    for num in nums:
        ans += abs(num)
    
    print(ans)

if __name__ == "__main__":
    main()
```

### Java
``` java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        
        int n = Integer.parseInt(br.readLine());
        StringTokenizer st = new StringTokenizer(br.readLine());
        
        long ans = 0;
        for (int i = 0; i < n; i++) {
            int num = Integer.parseInt(st.nextToken());
            ans += Math.abs(num);
        }
        
        System.out.println(ans);
    }
}
```

### Go
``` go
package main

import (
	"bufio"
	"fmt"
	"os"
)

func main() {
	const bufferSize = 1 << 20
	reader := bufio.NewReaderSize(os.Stdin, bufferSize)
	writer := bufio.NewWriterSize(os.Stdout, bufferSize)
	defer writer.Flush()

	var n int
	fmt.Fscan(reader, &n)

	ans := 0
	for i := 0; i < n; i++ {
		var num int
		fmt.Fscan(reader, &num)
		if num < 0 {
			ans -= num
		} else {
			ans += num
		}
	}

	fmt.Fprintln(writer, ans)
}
```
