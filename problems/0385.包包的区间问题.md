# 中位数贪心与绝对值等式变换
[题目链接](https://kamacoder.com/problempage.php?pid=1465)
## 思路分析
这道题目要求我们找出一个整数 x，使得所有区间到 x 的距离之和最小。
定义点 x 到区间 [l_i, r_i] 的距离函数为 f_x(i)。通过观察可知，这个距离可以用绝对值的形式统一表示：
f_x(i) = (|x - l_i| + |x - r_i| - (r_i - l_i)) / 2

那么，我们要最小化的总距离表达式可以转化为：
Sum = 1/2 * ( sum(|x - l_i|) + sum(|x - r_i|) ) - 1/2 * sum(r_i - l_i)

上式中，后半部分 1/2 * sum(r_i - l_i) 是常数（即所有区间长度和的一半），与 x 的取值无关。因此，要使总和 Sum 最小，我们只需要让 sum(|x - l_i|) + sum(|x - r_i|) 最小。

我们将所有的 l_i 和 r_i 放入同一个数组 P 中。数组 P 中一共包含 2n 个端点。
问题就转化为了经典的模型：**求数轴上一点 x 到 2n 个定点的距离之和最小**。
根据中位数定理，当 x 取这 2n 个点的中位数时，距离之和最小。由于点的总数是偶数（2n），x 取排序后第 n 个点到第 n+1 个点之间的任意值都可以获得最小值。

我们可以直接取排序后的第 n 个点作为最优的 x。
但实际上我们甚至不需要求出具体的 x。对于排好序的数组 P，最优的 x 将数组一分为二：前 n 个点都小于等于最优 x，后 n 个点都大于等于最优 x。
此时，这 2n 个点到最优 x 的距离总和等于：后 n 个点之和（记为 S_right） 减去 前 n 个点之和（记为 S_left）。

把这个结论代回最初的等式：
最小总和 Sum = 1/2 * (S_right - S_left) - 1/2 * (sum(r_i) - sum(l_i))
即：最优答案 = (S_right - S_left - sum_r + sum_l) / 2。

按照这个公式，我们只需对这 2n 个点进行一次排序，再计算对应的四个和即可，且全程都可以避免寻找具体的 x 从而简化代码逻辑。注意过程中数据可能非常大，需要使用 64 位整型来防止溢出。

## 复杂度分析
1. 时间复杂度：O(nlogn)；
2. 空间复杂度：O(n)。

## 代码实现
### Cpp
``` cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    // 优化输入输出
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n;
    if (!(cin >> n)) return 0;
    
    vector<long long> P(2 * n);
    long long sum_l = 0, sum_r = 0;
    
    for (int i = 0; i < n; ++i) {
        long long l, r;
        cin >> l >> r;
        sum_l += l;
        sum_r += r;
        P[2 * i] = l;
        P[2 * i + 1] = r;
    }
    
    // 对所有的 2n 个端点排序
    sort(P.begin(), P.end());
    
    long long S_left = 0, S_right = 0;
    for (int i = 0; i < n; ++i) {
        S_left += P[i];           // 前 n 个较小的点之和
        S_right += P[i + n];      // 后 n 个较大的点之和
    }
    
    // 直接套用公式求出最优解
    long long ans = (S_right - S_left - sum_r + sum_l) / 2;
    cout << ans << "\n";
    
    return 0;
}
```

### Python
``` python
import sys

def main():
    # 优化读取方式
    input_data = sys.stdin.read().split()
    if not input_data:
        return
        
    n = int(input_data[0])
    
    P = []
    sum_l = 0
    sum_r = 0
    
    idx = 1
    for _ in range(n):
        l = int(input_data[idx])
        r = int(input_data[idx+1])
        idx += 2
        
        sum_l += l
        sum_r += r
        P.append(l)
        P.append(r)
        
    P.sort()
    
    S_left = sum(P[:n])
    S_right = sum(P[n:])
    
    ans = (S_right - S_left - sum_r + sum_l) // 2
    print(ans)

if __name__ == '__main__':
    main()
```

### Java
``` java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.io.IOException;
import java.util.StringTokenizer;
import java.util.Arrays;

public class Main {
    public static void main(String[] args) throws IOException {
        // 优化输入
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String line = br.readLine();
        while (line != null && line.trim().isEmpty()) {
            line = br.readLine();
        }
        if (line == null) return;
        
        StringTokenizer st = new StringTokenizer(line);
        int n = Integer.parseInt(st.nextToken());
        
        long[] P = new long[2 * n];
        long sum_l = 0, sum_r = 0;
        
        for (int i = 0; i < n; i++) {
            if (!st.hasMoreTokens()) st = new StringTokenizer(br.readLine());
            long l = Long.parseLong(st.nextToken());
            
            if (!st.hasMoreTokens()) st = new StringTokenizer(br.readLine());
            long r = Long.parseLong(st.nextToken());
            
            sum_l += l;
            sum_r += r;
            P[2 * i] = l;
            P[2 * i + 1] = r;
        }
        
        Arrays.sort(P);
        
        long S_left = 0, S_right = 0;
        for (int i = 0; i < n; i++) {
            S_left += P[i];
            S_right += P[i + n];
        }
        
        long ans = (S_right - S_left - sum_r + sum_l) / 2;
        System.out.println(ans);
    }
}
```

### Go
``` go
package main

import (
	"bufio"
	"fmt"
	"os"
	"sort"
	"strconv"
)

func main() {
    // 采用 Scanner 优化大数据读入
	scanner := bufio.NewScanner(os.Stdin)
	scanner.Split(bufio.ScanWords)
    // 扩大 buffer 防止数据截断
	buf := make([]byte, 1024*1024*10)
	scanner.Buffer(buf, 1024*1024*10)

	if !scanner.Scan() {
		return
	}
	n, _ := strconv.Atoi(scanner.Text())

	P := make([]int, 2*n)
	var sum_l, sum_r int64 = 0, 0

	for i := 0; i < n; i++ {
		scanner.Scan()
		l, _ := strconv.Atoi(scanner.Text())
		scanner.Scan()
		r, _ := strconv.Atoi(scanner.Text())
		
		sum_l += int64(l)
		sum_r += int64(r)
		P[2*i] = l
		P[2*i+1] = r
	}

	sort.Ints(P)
	
	var S_left, S_right int64 = 0, 0
	for i := 0; i < n; i++ {
		S_left += int64(P[i])
		S_right += int64(P[i+n])
	}

	ans := (S_right - S_left - sum_r + sum_l) / 2
	fmt.Println(ans)
}
```
