# 区间dp
[题目链接](https://kamacoder.com/problempage.php?pid=1437)
## 思路分析

**状态定义**：dp[i][j]为开区间（i，j）内全部消除所能获得的最大分数。

**状态转移**：枚举区间（i，j）中的点k，假设我们最后一张抽走的卡片为k，那么就表明其他的卡片全部抽走了，而这个区间只剩一张卡片k，那么抽走k能获得的积分为c[i] * c[j]。因为这个区间只剩一张卡片k，所以与k相连的卡片只剩开区间的两个端点了。这时候dp[i][j]就可以拆分为dp[i][k]和dp[k][j]，以及c[i] * c[j]（这表示两边的卡片的分数乘积）。注意：dp[i][k]表示不包含两个端点i和k，将其中的其他卡片全抽走的最大积分，这与我们的假设不矛盾，由于k是最后一个抽走的，并且两端不能被抽走，所以这种拆分是合理的。同理dp[k][j]也是如此。

**转移方程**：

1. dp[i][j] = max(dp[i][j], dp[i][k] + dp[k][j] + c[i] * c[j]), i < k < j.

为了方便状态转移，我们可以手动给数组的首尾添加两个1，方便最后一张卡片抽走的积分计算。最终的答案就是dp[0][n + 1].

## 复杂度分析
1. 时间复杂度：O(n^3)；
2. 空间复杂度：O(n^2)。

### Python
``` python
import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    
    n = int(data[0])
    nums = [1] * (n + 2)
    for i in range(1, n + 1):
        nums[i] = int(data[i])
    
    dp = [[0] * (n + 2) for _ in range(n + 2)]
    
    for i in range(n, -1, -1):
        for j in range(i + 2, n + 2):
            for k in range(i + 1, j):
                dp[i][j] = max(dp[i][j], dp[i][k] + dp[k][j] + nums[i] * nums[j])
    
    print(dp[0][n + 1])

if __name__ == "__main__":
    main()
```

### Go
``` go
package main

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
	"strings"
)

func main() {
	scanner := bufio.NewScanner(os.Stdin)
	
	// 读取第一行：n
	scanner.Scan()
	n, _ := strconv.Atoi(scanner.Text())
	
	// 读取第二行：n个数字
	scanner.Scan()
	parts := strings.Fields(scanner.Text())
	
	nums := make([]int, n+2)
	nums[0] = 1
	nums[n+1] = 1
	for i := 1; i <= n; i++ {
		nums[i], _ = strconv.Atoi(parts[i-1])
	}
	
	dp := make([][]int, n+2)
	for i := range dp {
		dp[i] = make([]int, n+2)
	}
	
	for i := n; i >= 0; i-- {
		for j := i + 2; j <= n+1; j++ {
			for k := i + 1; k < j; k++ {
				val := dp[i][k] + dp[k][j] + nums[i]*nums[j]
				if val > dp[i][j] {
					dp[i][j] = val
				}
			}
		}
	}
	
	fmt.Println(dp[0][n+1])
}
```
