# 脑筋急转弯
[题目链接](https://kamacoder.com/problempage.php?pid=1429)
## 思路分析

题目的要求很简单，既要将数据分解为2的幂乘3的幂之和，还要选择3的幂最低的次数。那么也就是说我们可以直接不选择3，也就是说将所有的3的幂次全部指定为0，那么这时候我们的目的就变成了将一个数分解为2的幂次之和，那就是二进制分解了。

原题的要求并没有要求3的幂次最少，原题是输出任意一种方案即可。但是这样测评比较难，所以这里选择统一答案，但是按照原题的要求，似乎也是我们这种做法就可以，因为二进制分解是不会有重复元素的。

不知道出题人出这道题是想干嘛，也许只是脑筋急转弯吧。

## 复杂度分析
1. 时间复杂度：O(nlogC)；
2. 空间复杂度：O(1)。

### Python
``` python
import sys

def main():
    input = sys.stdin.readline
    t = int(input())
    out_lines = []
    
    for _ in range(t):
        n = int(input())
        ans = []
        idx = 0
        
        while n:
            if n & 1:
                ans.append(idx)
            n >>= 1
            idx += 1
        
        out_lines.append(str(len(ans)))
        # 反转并计算2的幂次
        result = []
        for power in reversed(ans):
            result.append(str(1 << power))
        out_lines.append(' '.join(result))
    
    sys.stdout.write('\n'.join(out_lines))

if __name__ == "__main__":
    main()
```

### Java
``` java
import java.io.*;
import java.util.*;

public class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));
        
        int t = Integer.parseInt(br.readLine());
        
        while (t-- > 0) {
            int n = Integer.parseInt(br.readLine());
            List<Integer> ans = new ArrayList<>();
            int idx = 0;
            
            while (n > 0) {
                if ((n & 1) == 1) {
                    ans.add(idx);
                }
                n >>= 1;
                idx++;
            }
            
            // 输出结果个数
            bw.write(ans.size() + "\n");
            
            // 反转并计算2的幂次
            if (!ans.isEmpty()) {
                for (int i = ans.size() - 1; i >= 0; i--) {
                    bw.write((1 << ans.get(i)) + (i > 0 ? " " : ""));
                }
                bw.newLine();
            }
        }
        
        bw.flush();
    }
}
```

### Go
``` go
package main

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
)

func main() {
	scanner := bufio.NewScanner(os.Stdin)
	scanner.Split(bufio.ScanWords)
	
	// 读取t
	scanner.Scan()
	t, _ := strconv.Atoi(scanner.Text())
	
	for ; t > 0; t-- {
		// 读取n
		scanner.Scan()
		n, _ := strconv.Atoi(scanner.Text())
		
		ans := make([]int, 0, 32)
		idx := 0
		
		for n > 0 {
			if n&1 == 1 {
				ans = append(ans, idx)
			}
			n >>= 1
			idx++
		}
		
		// 输出结果个数
		fmt.Println(len(ans))
		
		// 反转并计算2的幂次
		if len(ans) > 0 {
			// 先输出第一个元素
			fmt.Printf("%d", 1<<uint(ans[len(ans)-1]))
			
			// 输出剩余元素
			for i := len(ans) - 2; i >= 0; i-- {
				fmt.Printf(" %d", 1<<uint(ans[i]))
			}
			fmt.Println()
		}
	}
}
```
