# 动态规划
[题目链接](https://kamacoder.com/problempage.php?pid=1467)
## 思路分析
题目要求最大化匹配度。在调整阶段，当前元素要想最后匹配，只有以下几种情况：
1. 本身就匹配：ai = bi
2. 借用下一项的 a：ai = ai+1 （此时修改 bi 为 ai+1，使得 bi = ai）
3. 借用下一项的 b：bi = bi+1 （此时修改 ai 为 bi+1，使得 ai = bi）
4. 借用下一项的双值且下一项本身匹配：ai+1 = bi+1 （此时 ai 和 bi 都修改为下一项的值，从而匹配）

题目说，先**删除**（至少一次），然后在新数组里从前往后进行**调整**。调整的规则是：
- 修改 a_i 为 b_{i+1}；
- 修改 b_i 为 a_{i+1}。

我们不要正向去想“它变成了什么”，而是**逆向思考：“在新数组中，第 i 个位置最终能匹配（a'_i = b'_i），到底需要满足什么条件？”**

假设我们在原数组中挑出了一个子序列，其中有两个相邻的元素，它们在原数组的下标分别是 i 和 j （i < j）。在新数组里它们挨在一起。
此时，原数组的第 i 个元素想要达成匹配，有且仅有以下 **4 种途径**：

1. **自带光环**：原数组中 a_i = b_i。它不需要向后借元素，自己就匹配了。
2. **借用A**：如果 a_i = a_j，它可以把自己的 b_i 改成后一个元素的 a_j（即操作中的 b_i = a_{i+1}），从而使得 a_i = b_i 达成匹配。
3. **借用B**：如果 b_i = b_j，同理，它可以把自己的 a_i 改成后一个元素的 b_j，达成匹配。
4. **万能后继**：如果后一个元素 a_j = b_j，那么 i 可以把自己的 a 和 b **双双改成后一个元素的值**。因为 a_j = b_j，改完后自然就有 a'_i = b'_i 达成匹配。更妙的是，因为 j 本身是 a_j = b_j，j 不用做任何修改自己也能匹配！这就相当于 j 是个万能充电宝，前面挨着它的人都能因为吸了它的血而复活，同时它自己还不死。

**总结核心思路：**
我们要从原数组中挑出一个序列。对于序列中的前一个元素 i 连向后一个元素 j：
- 只要满足上述 2、3、4 之一，**本来不匹配的 i，就能获得 1 个匹配数。**
- 如果 i 本身就是满足条件 1（a_i = b_i），它本身就算 1 个匹配数，不需要通过 j 获得。

---

### 第二步：设计动态规划 (DP) 状态

有了上面的提炼，这就变成了一个典型的**最长子序列问题**。
我们定义 `dp[j]` 为：**以原数组中第 j 个元素作为子序列的结尾，能获得的最大匹配数。**

在转移时，我们要找一个排在它前面的 i (i < j)。
从 i 转移到 j 时，序列的总匹配数 = `dp[i]` + `(i 连到 j 产生的额外匹配)` + `(j 自己的自带匹配)`。

这里有一个极容易**重复计算**的陷阱：
如果 i 本身满足 a_i = b_i，那它在计算 `dp[i]` 时，**已经给自己加过 1 分了**。它向 j 转移时，不能再获得额外加分（一个位置最多提供 1 个匹配）。
所以，**只有当 a_i ≠ b_i 时（即 i 是个“不匹配的饥饿元素”），它才有可能因为连向了 j 而获得额外的 +1 分。**

---

### 第三步：代码中的核心变量

如果用两层 `for` 循环（O(n^2)）去找前面的最优 i，会超时。所以代码用几个变量（或哈希表）来**记录前面所有求过的 `dp[i]` 的最大值**，从而把查找优化到了 O(1)。

对应代码中的变量，我们来看看它们分别记录了什么：

1. **`M_all`**：前面所有的 i 中，`dp[i]` 的最大值。
   - **作用**：不管 i 和 j 满不满足上述的 2, 3, 4 条件，直接把 i 接到 j 前面（不产生额外加分）。这是最保底的转移。

2. **`M_unmatched`**：前面所有 **a_i ≠ b_i** 的元素中，`dp[i]` 的最大值。
   - **作用**：对应上面说的 **途径4（万能后继）**。当当前的 j 满足 a_j = b_j 时，前面的所有的“饥饿元素”都能因为接到 j 前面而复活。所以可以直接取 `M_unmatched + 1`。

3. **`max_dp_a[x]`**：前面所有 **a_i ≠ b_i** 且 **a_i = x** 的元素中，`dp[i]` 的最大值。
   - **作用**：对应 **途径2（借用A）**。如果当前的 a_j = x，那前面记录在这里的“饥饿元素”就可以借用它复活，得分 `max_dp_a[x] + 1`。

4. **`max_dp_b[y]`**：前面所有 **a_i ≠ b_i** 且 **b_i = y** 的元素中，`dp[i]` 的最大值。
   - **作用**：对应 **途径3（借用B）**。同理，得分 `max_dp_b[y] + 1`。



这相当于我们在原数组中挑选出一个长度为 m 的子序列（由于至少删除一次，要求 m ≤ n-1）。
我们可以用动态规划来求解，设 `dp[j]` 表示以原数组第 j 个元素为子序列的最后一个元素时，所能得到的 **最大匹配数** 以及在达到该匹配数下的 **最少元素使用个数**。
我们每次让 j 接在某个 i < j 的后面时，如果 ai ≠ bi，只要满足上述 2、3、4 的条件之一，便可以在原匹配数基础上增加 1。

通过维护当前所有前缀的最大状态，可以把寻找最优前驱的过程优化为 O(1)。
最后，如果全局能达到的最大匹配数所必须使用的最少元素个数等于 n（即必须保留全数组才能达到该最大匹配），受限于 m ≤ n-1，答案即为 `最大匹配数 - 1`；否则，答案就等于 `最大匹配数`。

## 复杂度分析
1. 时间复杂度：O(nlogn) 或 O(n)，取决于使用了树形字典还是哈希表/离散化。C++ 采用离散化优化可以做到极小的常数，总时间复杂度为 O(nlogn)。
2. 空间复杂度：O(n)，用于存储状态数组和哈希/离散化数组。

``` cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

void solve() {
    int n;
    cin >> n;
    vector<int> a(n), b(n);
    vector<int> vals;
    vals.reserve(2 * n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        vals.push_back(a[i]);
    }
    for (int i = 0; i < n; i++) {
        cin >> b[i];
        vals.push_back(b[i]);
    }
    
    // 离散化优化时间
    sort(vals.begin(), vals.end());
    vals.erase(unique(vals.begin(), vals.end()), vals.end());
    int num_vals = vals.size();
    
    auto get_id = [&](int x) {
        return lower_bound(vals.begin(), vals.end(), x) - vals.begin();
    };
    
    pair<int, int> M_all = {-1, -1};
    pair<int, int> M_unmatched = {-1, -1};
    vector<pair<int, int>> max_dp_a(num_vals, {-1, -1});
    vector<pair<int, int>> max_dp_b(num_vals, {-1, -1});
    
    for (int j = 0; j < n; j++) {
        int aj = get_id(a[j]);
        int bj = get_id(b[j]);
        
        pair<int, int> best_prev = {0, 0};
        if (M_all.first != -1) {
            best_prev = max(best_prev, M_all);
        }
        
        if (aj == bj) {
            if (M_unmatched.first != -1) {
                best_prev = max(best_prev, {M_unmatched.first + 1, M_unmatched.second});
            }
        } else {
            if (max_dp_a[aj].first != -1) {
                best_prev = max(best_prev, {max_dp_a[aj].first + 1, max_dp_a[aj].second});
            }
            if (max_dp_b[bj].first != -1) {
                best_prev = max(best_prev, {max_dp_b[bj].first + 1, max_dp_b[bj].second});
            }
        }
        
        pair<int, int> dp_j = {best_prev.first + (aj == bj ? 1 : 0), best_prev.second - 1};
        
        M_all = max(M_all, dp_j);
        if (aj != bj) {
            M_unmatched = max(M_unmatched, dp_j);
            max_dp_a[aj] = max(max_dp_a[aj], dp_j);
            max_dp_b[bj] = max(max_dp_b[bj], dp_j);
        }
    }
    
    int best_match = M_all.first;
    int best_elements = -M_all.second;
    
    if (best_elements == n) {
        cout << best_match - 1 << "\n";
    } else {
        cout << best_match << "\n";
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int t;
    if (cin >> t) {
        while (t--) {
            solve();
        }
    }
    return 0;
}
```

``` python
import sys

def solve():
    input_data = sys.stdin.read().split()
    if not input_data:
        return
    
    t = int(input_data[0])
    idx = 1
    out = []
    
    for _ in range(t):
        n = int(input_data[idx])
        idx += 1
        a = input_data[idx : idx + n]
        idx += n
        b = input_data[idx : idx + n]
        idx += n
        
        M_all = (-1, -1)
        M_unmatched = (-1, -1)
        max_dp_a = {}
        max_dp_b = {}
        
        for j in range(n):
            aj = a[j]
            bj = b[j]
            
            best_prev = (0, 0)
            if M_all[0] != -1 and M_all > best_prev:
                best_prev = M_all
                
            if aj == bj:
                if M_unmatched[0] != -1:
                    cand = (M_unmatched[0] + 1, M_unmatched[1])
                    if cand > best_prev:
                        best_prev = cand
            else:
                if aj in max_dp_a:
                    cand = (max_dp_a[aj][0] + 1, max_dp_a[aj][1])
                    if cand > best_prev:
                        best_prev = cand
                if bj in max_dp_b:
                    cand = (max_dp_b[bj][0] + 1, max_dp_b[bj][1])
                    if cand > best_prev:
                        best_prev = cand
            
            dp_j = (best_prev[0] + (1 if aj == bj else 0), best_prev[1] - 1)
            
            if dp_j > M_all:
                M_all = dp_j
            
            if aj != bj:
                if dp_j > M_unmatched:
                    M_unmatched = dp_j
                if aj not in max_dp_a or dp_j > max_dp_a[aj]:
                    max_dp_a[aj] = dp_j
                if bj not in max_dp_b or dp_j > max_dp_b[bj]:
                    max_dp_b[bj] = dp_j
                    
        best_match = M_all[0]
        best_elements = -M_all[1]
        
        if best_elements == n:
            out.append(str(best_match - 1))
        else:
            out.append(str(best_match))
            
    print('\n'.join(out))

if __name__ == '__main__':
    solve()
```

``` java
import java.io.InputStream;
import java.io.IOException;
import java.util.HashMap;

public class Main {
    static class FastScanner {
        private InputStream stream;
        private byte[] buf = new byte[1024];
        private int head, tail;

        public FastScanner(InputStream stream) {
            this.stream = stream;
        }

        int read() {
            if (head >= tail) {
                head = 0;
                try {
                    tail = stream.read(buf, 0, buf.length);
                } catch (IOException e) {
                    return -1;
                }
                if (tail <= 0) return -1;
            }
            return buf[head++];
        }

        int nextInt() {
            int c = read();
            while (c <= 32) {
                if (c == -1) return -1;
                c = read();
            }
            int res = 0;
            while (c > 32) {
                res = res * 10 + c - '0';
                c = read();
            }
            return res;
        }
    }

    static long pack(int match, int elementsNeg) {
        long m = match;
        long e = elementsNeg + 200000;
        return (m << 32) | e;
    }

    public static void main(String[] args) {
        FastScanner sc = new FastScanner(System.in);
        int t = sc.nextInt();
        if (t == -1) return;
        
        StringBuilder sb = new StringBuilder();
        while (t-- > 0) {
            int n = sc.nextInt();
            int[] a = new int[n];
            int[] b = new int[n];
            for (int i = 0; i < n; i++) a[i] = sc.nextInt();
            for (int i = 0; i < n; i++) b[i] = sc.nextInt();
            
            long M_all = -1L;
            long M_unmatched = -1L;
            HashMap<Integer, Long> max_dp_a = new HashMap<>();
            HashMap<Integer, Long> max_dp_b = new HashMap<>();
            
            for (int j = 0; j < n; j++) {
                int aj = a[j];
                int bj = b[j];
                
                long best_prev = pack(0, 0);
                if (M_all != -1L && M_all > best_prev) {
                    best_prev = M_all;
                }
                
                if (aj == bj) {
                    if (M_unmatched != -1L) {
                        int m = (int)(M_unmatched >> 32);
                        int e = (int)(M_unmatched & 0xFFFFFFFFL) - 200000;
                        long cand = pack(m + 1, e);
                        if (cand > best_prev) best_prev = cand;
                    }
                } else {
                    if (max_dp_a.containsKey(aj)) {
                        long val = max_dp_a.get(aj);
                        int m = (int)(val >> 32);
                        int e = (int)(val & 0xFFFFFFFFL) - 200000;
                        long cand = pack(m + 1, e);
                        if (cand > best_prev) best_prev = cand;
                    }
                    if (max_dp_b.containsKey(bj)) {
                        long val = max_dp_b.get(bj);
                        int m = (int)(val >> 32);
                        int e = (int)(val & 0xFFFFFFFFL) - 200000;
                        long cand = pack(m + 1, e);
                        if (cand > best_prev) best_prev = cand;
                    }
                }
                
                int prev_m = (int)(best_prev >> 32);
                int prev_e = (int)(best_prev & 0xFFFFFFFFL) - 200000;
                long dp_j = pack(prev_m + (aj == bj ? 1 : 0), prev_e - 1);
                
                if (dp_j > M_all) M_all = dp_j;
                if (aj != bj) {
                    if (dp_j > M_unmatched) M_unmatched = dp_j;
                    if (!max_dp_a.containsKey(aj) || dp_j > max_dp_a.get(aj)) {
                        max_dp_a.put(aj, dp_j);
                    }
                    if (!max_dp_b.containsKey(bj) || dp_j > max_dp_b.get(bj)) {
                        max_dp_b.put(bj, dp_j);
                    }
                }
            }
            
            int best_match = (int)(M_all >> 32);
            int best_elements = -((int)(M_all & 0xFFFFFFFFL) - 200000);
            
            if (best_elements == n) {
                sb.append(best_match - 1).append("\n");
            } else {
                sb.append(best_match).append("\n");
            }
        }
        System.out.print(sb.toString());
    }
}
```

``` go
package main

import (
	"bufio"
	"fmt"
	"os"
	"strconv"
)

type Pair struct {
	first  int
	second int
}

func maxPair(a, b Pair) Pair {
	if a.first > b.first {
		return a
	}
	if a.first == b.first && a.second > b.second {
		return a
	}
	return b
}

func solve() {
	scanner := bufio.NewScanner(os.Stdin)
	scanner.Split(bufio.ScanWords)
	out := bufio.NewWriter(os.Stdout)
	defer out.Flush()

	if !scanner.Scan() {
		return
	}
	t, _ := strconv.Atoi(scanner.Text())

	for iter := 0; iter < t; iter++ {
		scanner.Scan()
		n, _ := strconv.Atoi(scanner.Text())

		a := make([]int, n)
		b := make([]int, n)
		for i := 0; i < n; i++ {
			scanner.Scan()
			a[i], _ = strconv.Atoi(scanner.Text())
		}
		for i := 0; i < n; i++ {
			scanner.Scan()
			b[i], _ = strconv.Atoi(scanner.Text())
		}

		M_all := Pair{-1, -1}
		M_unmatched := Pair{-1, -1}
		max_dp_a := make(map[int]Pair)
		max_dp_b := make(map[int]Pair)

		for j := 0; j < n; j++ {
			aj := a[j]
			bj := b[j]

			best_prev := Pair{0, 0}
			if M_all.first != -1 {
				best_prev = maxPair(best_prev, M_all)
			}

			if aj == bj {
				if M_unmatched.first != -1 {
					best_prev = maxPair(best_prev, Pair{M_unmatched.first + 1, M_unmatched.second})
				}
			} else {
				if val, ok := max_dp_a[aj]; ok {
					best_prev = maxPair(best_prev, Pair{val.first + 1, val.second})
				}
				if val, ok := max_dp_b[bj]; ok {
					best_prev = maxPair(best_prev, Pair{val.first + 1, val.second})
				}
			}

			add := 0
			if aj == bj {
				add = 1
			}
			dp_j := Pair{best_prev.first + add, best_prev.second - 1}

			M_all = maxPair(M_all, dp_j)
			if aj != bj {
				M_unmatched = maxPair(M_unmatched, dp_j)
				if val, ok := max_dp_a[aj]; !ok || maxPair(val, dp_j) == dp_j {
					max_dp_a[aj] = dp_j
				}
				if val, ok := max_dp_b[bj]; !ok || maxPair(val, dp_j) == dp_j {
					max_dp_b[bj] = dp_j
				}
			}
		}

		best_match := M_all.first
		best_elements := -M_all.second

		if best_elements == n {
			fmt.Fprintln(out, best_match-1)
		} else {
			fmt.Fprintln(out, best_match)
		}
	}
}

func main() {
	solve()
}
```
